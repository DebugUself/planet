现象¶

  • gitlab.com 每天不同时段打开的速度完全不同
  • github-pages 发布的静态网站经常无法打开
  • git pull 工作仓库有时说仓库不存在
  • …

  • 移动宽带, 手机158套餐送的, 很多年了, 也自动升级到 500M 光纤, 配 Nighthawk R7000 无线路由器

  • 华为 mifi , 自驾旅游时用, 配置飞猫物联网流量卡, 600G/月
