宇宙间最大图书馆

将所有图书文字加密, 编辑为一个连续数字, 作为一个无理小数本体;

然后,找到这个无理数的精确整数比, 类似 113/355 这种,

然后, 将这个比例, 刻为一个稳定材料制造的小棍上, 那么, 整个儿图书馆的所有内容, 就变成了一个小棍物理外观上携带的一个点儿;

到任何稳定环境中, 只要精确测量出来那个点所在位置分割小棍的比例, 一除就获得了所有加密数字, 再根据解密原则就能还原出所有内容了,
